除了一些幾乎所有 IDE 都有基本快速鍵外，這邊筆記一些使用 vs code 不一樣，或者新發現的快速鍵。

# 功能
- Ctrl + P: 搜尋檔案
- Ctrl + Shift + P: Command Palette 指令功能列
- Ctrl + `: 開啟 cmd

# 編輯
- Ctrl + D: 同文件中同字串選取
- Alt + Shift + F: 文件排版格式化
- Alt + Shift + 下: 複製當前行
- Alt + Shift + 滑鼠左鍵: 跨行選取
